"# Projet-Canal_Plus"
 
Tutoriel

N'ayant pas le chemin de la route dans l'exercice. J'ai réalisé le test en front.

Après avoir chargé le projet du repository et l'avoir ouvert dans IDE, vous devrez vous déplacer au dossier src/test/java/modules. 

Vous trouverez dans ce dossier la class test Runner où vous pourrez lancer les tests; soit en click droit sur la phrase TestRunner puis click sur Run testRunner; soit avec le raccourci CTRL+MAJ+F10.

Une fois le test terminé, vous allez voir que l'adresse ne se met pas à jour, il faut créer un ticket et l'envoyer au développeur afin qu'il corrige ce bug.

Bonne consulatation du code.
